What is an enumerable?

An enumerable basically allows us to count through a collection of data and sort it in some way.

The group_by method works by allowing you to form groups based on the rules that you decide.

It then returns a hash where the keys are the answer to the set rules and the values are arrays of elements from the original data that follow those rules, and therefore, are linked to the keys.

It's syntax looks like this:

group_by {|obj| block} --> a_hash

An example would be...

(1..6).group_by {|x| x % 3 }

will ouput...

{ 0 => [3,6], 1 => [1,4], 2 => [2,5] }

In the example above, the numbers 1 through 6 were grouped by the rule:

For every number in the set, group it by what the remainder is when that number is divided by 3

As the output reads, the key 0 is assigned the values 3 and 6. This means that the numbers 3 and 6, when divided by 3, give a remainder of 0, which is why they are both under the same key (0).

group_by is a great tool to use when you want to sort through a set of data and group it according to a set of rules that you choose!
